# torn-api
`torn-api` is a lightweight proxy that calls the Torn API and serves the results as JSON through a simple web interface. It helps Torn players access API data while keeping their personal API key hidden. Built for simplicity and speed — simple to run, free to use, and fast to respond.

## Tagline:
Simple. Free. Fast.

## Key Features
- Fetches data from the Torn API
- Returns clean JSON responses
- Conceals the user's Torn API key
- Simple web interface
- Lightweight and fast

## Target Users
Torn players who want to safely access or share API-based data without exposing their personal API key.
